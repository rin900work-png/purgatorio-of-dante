# Данте. «Божественная комедия»: Чистилище — разбор по песням

Маленький **статический сайт**: краткое описание сюжета и главных моментов второй
кантики поэмы Данте Алигьери — «Чистилище» (*Purgatorio*) — по всем 33 песням.
Отдельный проект, никак не связанный с другими репозиториями.

Стек намеренно простой — **без Flask, без сервера, без зависимостей**: чистый
Python-генератор собирает HTML, который можно открыть локально или бесплатно
захостить как статику.

К каждой песне и к главным персонажам подобрана иллюстрация в **общественном
достоянии** (источник — Wikimedia Commons): большинство песен — гравюры **Гюстава
Доре** (1868); семь песен, которых Доре не делал (6, 11, 14, 17, 21, 22, 26), —
рисунки **Сандро Боттичелли** (1480-е). Персонажи: портрет Данте кисти Боттичелли,
античная римская мозаика с Вергилием (Музей Бардо, Тунис), «Данте и Беатриче»
Генри Холидея, Катон и Стаций — Доре. Файлы лежат в `static/img/`, привязка — в
конце `content.py` (`_SONG_IMG` и `_CHAR_IMG`).

## Структура

```
content.py        — все данные (текст разбора): мета, 3 части, лица, темы, 33 песни
build.py          — генератор: читает content.py → собирает HTML в docs/
static/style.css  — стили (тема «рассвет над горой»: пергамент + зелень надежды)
docs/             — СГЕНЕРИРОВАННЫЙ сайт (index.html, song-01..33.html, style.css)
```

Хочешь поправить текст разбора — правишь `content.py` и пересобираешь. Менять HTML
руками не нужно.

## Сборка

Нужен только Python 3 (стандартная библиотека, никаких `pip install`):

```bash
python3 build.py
```

Команда заново собирает папку `docs/`. Посмотреть локально — просто открыть
`docs/index.html` в браузере.

## Композиция (3 части)

- **I. Предчистилище** (песни 1–9) — берег и склоны горы: поздно раскаявшиеся,
  отлучённые, нерадивые государи и врата собственно Чистилища.
- **II. Семь кругов горы** (песни 10–27) — семь уступов, где искупаются смертные
  грехи от гордыни внизу до сладострастия у вершины: гордыня, зависть, гнев,
  уныние, скупость, чревоугодие, похоть.
- **III. Земной рай** (песни 28–33) — Эдемский лес на вершине: Мательда,
  мистическая процессия, явление Беатриче, исповедь и реки Лета и Эвноэ.

## Как опубликовать (GitHub Pages)

Сайт — статика, сервер не нужен. Заводится отдельный репозиторий, в Settings → Pages
указывается ветка `main`, папка `/docs`. После правок:

```bash
python3 build.py        # пересобрать docs/
git add -A
git commit -m "обновил разбор"
git push                # GitHub Pages подхватит изменения за ~минуту
```

Папку `docs/` так же можно раздать на Cloudflare Pages, Netlify или Vercel.
Понадобится бэкенд — брать serverless (Cloudflare Workers / Vercel functions),
а не выделенный сервер.
